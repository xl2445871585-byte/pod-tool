# POD产品批量上架工具

一个简单易用的POD（Print on Demand）产品批量上架工具，支持自动处理图片、上传图床、生成店小秘导入Excel。

## 功能特点

- ✅ 支持ZIP压缩包或单张图片上传
- ✅ 自动识别图片命名规则
- ✅ 自动上传图片到图床（ImgBB）
- ✅ 自定义尺寸价格配置
- ✅ 一键生成店小秘导入Excel
- ✅ 无需编程知识，打开网页就能用

## 快速开始

### 方式1：在线使用（推荐）
访问部署好的网站地址即可使用

### 方式2：本地运行
```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```

## 图片命名规则

支持两种格式：

1. **款号-图片编号**：如 `a-1.jpg`, `a-2.jpg`
2. **PS导出格式**：如 `未标题-1(1)_01.jpg`

## 默认尺寸配置

| 编号 | 尺寸 | 价格 |
|-----|------|------|
| 01 | 40*60cm | 100元 |
| 02 | 40*120cm | 190元 |
| 04 | 80*120cm | 240元 |

可在网页左侧自定义修改。

## 技术栈

- Streamlit - 网页框架
- Pandas - 数据处理
- ImgBB - 图床服务
